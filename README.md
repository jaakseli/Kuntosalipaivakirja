# Kuntosalipaivakirja

Sovelluksessa käyttäjät pystyvät pitämään kirjaa omista kuntosalitreeneistään ja jakamaan niitä toisille käyttäjille.
Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
Käyttäjä pystyy lisäämään treenejä ja muokkaamaan ja poistamaan niitä, sekä julkaisemaan omia treenisuorituksiaan.
Käyttäjä näkee muiden käyttäjien treenit, mikäli treeni on merkitty julkiseksi.
Käyttäjä pystyy kommentoimaan muiden käyttäjien julkisia treenejä
Käyttäjäsivu näyttää, montako treeniä käyttäjä on lisännyt ja listan käyttäjän lisäämistä treeneistä.
